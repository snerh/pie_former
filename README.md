# pie_former
Simle project to study pygame features.

You need pygame package installed in your system or python virtual env.

If you have system-wide intalled package pygame just run game (skip steps 1-3).

1. Create a virtual environment using:

```python -m venv ./venv```

2. Activate virtual environment:

```source ./venv/bin/activate```

3. Install pygame:

``` pip install pygame```

4. Run game:

```python src/main.py```

5. PROFIT

Bye!
